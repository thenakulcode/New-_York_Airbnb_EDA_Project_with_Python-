# New-_York_Airbnb_EDA_Project_with_Python-
Exploratory data analysis and predictive modeling for New York Airbnb listings. Cleans data, engineers features, visualizes trends, and trains regression models to predict prices using pandas, NumPy, scikit-learn, and seaborn. Notebook and dataset included; workflows, evaluation, and visuals. Includes EDA, feature selection, tuning, CV, and charts
